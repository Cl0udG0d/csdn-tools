# 发表博客的工具

## 为什么有它?
我在很多网站上都有博客,当发表博客时,每个网站都要更新,太累了,为了避免重复操作,所以想b通过程序自动发博客
同时, 这个repo 还有其他一些小工具, 比如 picPost可以检查文章中的本地图片链接, 然后上传到七牛云

## 需要些什么?
### blog-sender
* python3
* requests
### picPost
* qshell: 七牛云的一个命令行工具
* 七牛云账号
